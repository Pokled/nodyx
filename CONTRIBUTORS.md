## Want to contribute?

1. Browse [open Issues](https://github.com/Pokled/nodyx/issues) — look for the `good first issue` label
2. Read [CONTRIBUTING.md](docs/en/CONTRIBUTING.md)
3. Open a PR — the bar is "does it work and does it respect the existing patterns", not "is it perfect"

We merge-then-polish rather than gatekeep. If your code has rough edges, we'll add a follow-up commit and credit you here either way.

— The Nodyx project
